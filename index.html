Projects

PressNotify

PressNotify is a tool that allows you to automatically send pitch e-mails to various publications using a template provided. Just fill out the form and press send! I developed this tool at Promediacorp while working as a software engineer intern in summer 2017.

Pass The Turing

PassTheTuring (Alan) is a bot that analyzes movie scripts and finds the most appropriate response based on previous interactions between characters in a movie. We fed it over 100 movie scripts (ranging from 12 Angry Men to The Dark Night).

Up Hail

Up Hail is a project we developed at Promediacorp, Inc. It is a tool that allows users to find the best rate between different service providers like Uber and Lyft.

ZTSRating.com

ZTSRating.com is a project I developed that allows users to find the 0-60 time of their vehicle or other vehicles.

Task Management Tool for USA Reconditioning

This task management tool is a project I developed on a freelance basis for a helmet reconditioning company. It manages employees and assigns tasks to different workers and tracks their progress.

UCI Firewall

We created this at a security hackathon at UCI. It tracked who is in which room by detecting their wifi access point. It was meant to detect intruders that had devices connected to an access point in a room they are not supposed to be in.
